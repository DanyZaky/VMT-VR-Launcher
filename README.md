# VR Launcher 🚀

Welcome to **VR Launcher**! This application is specifically designed to make it incredibly easy for you to manage and update your Virtual Reality (VR) games/applications without worrying about losing important data or going through complicated installation processes.

## ✨ Key Features

- **One-Click Updates:** Update your VR application simply by selecting the latest update package and clicking a single button.
- **Multi-Format Support:** No manual extraction needed! The launcher natively reads and processes update packages in **.zip**, **.rar**, and **.7z** formats.
- **Smart Data Preservation:** Afraid of losing your save files? This launcher automatically backs up and preserves your valuable files (such as `DataSave`, network configurations, and files inside the *StreamingAssets* folder) every time an update occurs.
- **Real-time Information Panel:** Instantly view a summary of your application's status:
  - **Build Info:** Game Name, Version, and Build Date.
  - **Network Telemetry:** Connection status, Server, Port, and Database URL.
- **Elegant Interface:** Designed with a modern, clean, and eye-friendly Dark Mode interface.

## 📥 How to Use

1. **Set the Game Location (Build Directory):** 
   On the top panel, click **Browse** to select the folder where your VR application/game is currently installed.
2. **Select the Update File (Update Package):** 
   On the panel below, click **Browse** to select the latest update package (`.zip`, `.rar`, or `.7z`).
3. **Start the Update:** 
   Click the blue **UPDATE BUILD** button. Sit back and wait for the loading bar to finish. Your application will be updated while your save data remains completely safe!
4. **Play:** 
   Once the update is complete, click the **INITIALIZE APP** button in the bottom right corner to jump straight into the VR world.

## 🛠️ Extra Tools

- **SYNC DATA:** Use this to refresh the network and version information if you made manual changes to the game's folder.
- **OPEN DIRECTORY:** A quick shortcut to open your game's installation folder directly in Windows Explorer.
